config {PlantMAT} R Documentation

create plantMAT configuration

Description


if all of the parameter is omit, then you can create a settings
model with all configuration set to default values.

Usage

config(
    AglyconeType = All,
    AglyconeSource = All,
    AglyconeMWRange = [600],
    NumofSugarAll = [6],
    NumofAcidAll = [1],
    NumofSugarHex = [6],
    NumofSugarHexA = [6],
    NumofSugardHex = [6],
    NumofSugarPen = [6],
    NumofAcidMal = [1],
    NumofAcidCou = [1],
    NumofAcidFer = [1],
    NumofAcidSin = [1],
    NumofAcidDDMP = [1],
    PrecursorIonType = ['[M]-','[M+H]+','[M-H]-'],
    SearchPPM = 10,
    NoiseFilter = 0.05,
    mzPPM = 30,
    aglyconeSet = NULL);
