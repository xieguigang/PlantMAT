config {PlantMAT} R Documentation

create plantMAT configuration

Description


if all of the parameter is omit, then you can create a settings
model with all configuration set to default values.

Usage

config(
    AglyconeType = All,
    AglyconeSource = All,
    AglyconeMWRange = [400,600],
    NumofSugarAll = [0,6],
    NumofAcidAll = [0,1],
    NumofSugarHex = [0,6],
    NumofSugarHexA = [0,6],
    NumofSugardHex = [0,6],
    NumofSugarPen = [0,6],
    NumofAcidMal = [0,1],
    NumofAcidCou = [0,1],
    NumofAcidFer = [0,1],
    NumofAcidSin = [0,1],
    NumofAcidDDMP = [0,1],
    PrecursorIonType = ["[M]+","[M]-","[M+H]+","[M-H]-"],
    SearchPPM = 10,
    NoiseFilter = 0.05,
    mzPPM = 30,
    aglyconeSet = NULL);
